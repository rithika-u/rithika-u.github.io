Notes on current status

✅ The repository now contains a fully functional Pac-Man Valentine's game and arXiv papers page! Both are live and playable on GitHub Pages. The project includes GitHub Actions automation for nightly paper updates.

How It Works

  • Game Page: Pure JavaScript with HTML5 Canvas. Play with arrow keys or WASD, use Space to start. Eat roses for temporary heart-shooting power!
  • Papers Page: Fetches latest arXiv papers client-side with search, filtering, and sorting. Updates hourly when you visit; GitHub Actions auto-updates nightly.
  • Responsive Design: Both pages work great on desktop and mobile devices.
